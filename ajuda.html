Guia rápido

Como usar o sistema

Dashboard

Mostra visão geral da retífica, serviços em andamento, valores a receber e pendências.

Nova OS

Cadastra uma nova ordem de serviço com cliente, veículo, peça, serviço, valores e previsão.

Serviços

Lista todas as OS, permite editar, imprimir, enviar WhatsApp, alterar status e marcar como pago.

Clientes

Mostra clientes agrupados por telefone e histórico completo de serviços.

Financeiro

Controla recebido, pendente, OS pagas, OS parciais, exportação de relatórios e impressão.

Configurações

Personaliza nome da empresa, telefone, endereço, CNPJ, logo e dados da impressão.

Backup

Permite exportar e importar os dados do sistema para evitar perda.

Operação

Fluxo recomendado

  1. Cadastrar OS
  2. Atualizar status do serviço
  3. Avisar cliente pelo WhatsApp
  4. Marcar pagamento
  5. Consultar histórico
  6. Exportar relatório ou imprimir quando precisar
  7. Fazer backup regularmente

Atenção

Avisos importantes

  • Enquanto o sistema estiver local, os dados ficam salvos neste navegador.
  • Use o backup para guardar uma cópia dos dados.
  • Antes de trocar de computador ou navegador, exporte um backup.
  • Futuramente o sistema pode ser conectado à nuvem com login.
